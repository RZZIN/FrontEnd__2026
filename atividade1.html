Conceitos Básicos em Portugol

Portugol é uma linguagem de pseudocódigo utilizada para ensinar lógica de programação.

1 - Estrutura Sequencial

Executa as instruções uma após a outra, na ordem em que aparecem.

algoritmo "soma"
inicio
   escreva("Digite um número: ")
   leia(num1)
   escreva("Digite outro número: ")
   leia(num2)
   soma <- num1 + num2
   escreva("Resultado: ", soma)
fim

2 - Estrutura Condicional (SE / SENAO)

Utilizada para tomar decisões dentro do algoritmo.

algoritmo "media"
inicio
   leia(media)
   se media >= 7 entao
      escreva("Aprovado")
   senao
      escreva("Reprovado")
   fimse
fim

3 - Estrutura de Repetição (ENQUANTO / PARA)

Utilizada quando precisamos repetir uma ação várias vezes.

algoritmo "contagem"
inicio
   para i de 1 ate 5 faca
      escreva(i)
   fimpara
fim
